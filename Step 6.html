Step 6 - Add remote Origins

Once you create that repository, you will be greeted with this screen. we know it looks scary, but we will get through this together.

A little more than halfway down, there will be an option that says: "...or push an existing repository from the command line" which will have two lines of code under it.

Just like the picture shows, you can either BOTH of those lines of code, or you can simply click on the COPY icon located on the right-hand side of that option's container.

Commands used so far:

git init git status git add git commit
