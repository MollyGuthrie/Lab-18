During development, you will never be done writing code or making changes. The beautiful part is that if you have made any changes to your project, all we need to do is revisit a few commands...

git status git add git commit git push
Those four commands will check your status, add your files to tracking, make a new commit, and then push those changes to GitHub.com


Commands used so far:

git init git status git add git commit git push
